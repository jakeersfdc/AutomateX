## 🆕 **WHAT'S NEW IN v2.1**

### **3 Professional Indicators Added**

#### 1️⃣ ☁️ **Ichimoku Cloud** (STRONGEST)
```
Score Impact: ±3 (Maximum strength)

Signals:
✅ Above bull cloud + Tenkan > Kijun = +3 BUY
❌ Below bear cloud + Tenkan < Kijun = -3 SELL
⚡ Tenkan crosses Kijun = ±2 entry trigger
🟡 In cloud = consolidation, avoid

Advantage: All-in-one visual system, 4 indicators in 1
Best For: Identifying market direction at a glance
```

#### 2️⃣ ⚡ **Stochastic RSI** (FASTEST)
```
Score Impact: ±2 for oversold/overbought

Signals:
📉 K < 20 & D < 20 = +2 OVERSOLD (buy bounce)
📈 K > 80 & D > 80 = -2 OVERBOUGHT (sell pullback)
⚡ K > D = +1 bull cross
⚡ K < D = -1 bear cross

Advantage: Catches reversals 2-3 bars EARLIER than RSI
Best For: Intraday scalping (5-15 min)
```

#### 3️⃣ 📊 **Rate of Change (ROC)** (MOMENTUM FILTER)
```
Score Impact: ±2 for strong moves, 0 kills false signals

Signals:
💪 ROC > +1.5% = +2 STRONG bullish momentum
💪 ROC < -1.5% = -2 STRONG bearish momentum
💤 ROC near 0% = 0 points (KILLS ENTRY - momentum dead!)

Advantage: Prevents whipsaw entries when consolidating
Best For: Filtering weak signals in range-bound markets
```

### **Automatic Signal Scoring**
```
Instead of: "Is this a buy? Maybe... unclear"
v2.1: "Bull Score: 7/11 points = STRONG BUY"

Scoring:
- Ichimoku: ±3 (strongest)
- StochRSI: ±2 (fast entries)
- ROC: ±2 (momentum)
- RSI: ±1 (basic)
- MACD: ±1 (trend)
- Volume: ±1 (conviction)
- VIX: ±1 (volatility)
= Total: ±11 maximum

Rule:
✅ BUY if Bull Score >= 3
❌ SELL if Bear Score <= -3
🟡 NO TRADE if both < 2 (yellow zone)
```

4. Pine Script Editor → New Indicator

7. Add to Chart
```

### **Step 2: Configure**
```
Settings → Inputs:
├─ General: All ON ✓
├─ Ichimoku: TK 9, KJ 26, SEN 52 ✓
├─ StochRSI: Length 14, Smooth 3 ✓
├─ ROC: Length 9, Strong 1.5% ✓
├─ VIX: ON, Threshold 25 ✓
└─ OK
```

### **Step 3: Trade**
```

✅ Green arrow + Bull Score >= 3 = BUY

Set:
- SL: 0.5-1% away
