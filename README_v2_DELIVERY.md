## KEY SYSTEM COMPONENTS

### Support & Resistance Zones
```
R2 (Red)     → Sell target zone
R1 (Orange)  → Minor resistance / Sell opportunity
PIVOT (Blue) → Daily balance point
S1 (Teal)    → BUY opportunity / Minor support
S2 (Green)   → Strong support / Best BUY zone
```

### Signal Generation (Proven in Market)

**3 BUY Setups:**
1. **Oversold Bounce** - RSI < 30 at S1 + Volume + MACD up
2. **Pivot Breakout** - Close above Pivot + MACD up + Volume
3. **Volume Spike** - RSI < 35 at S2 + 1.5x volume + MACD up

**3 SELL Setups:**
1. **Overbought Reversal** - RSI > 70 at R1 + Volume + MACD down
2. **Pivot Breakdown** - Close below Pivot + MACD down + Volume
3. **Volume Spike** - RSI > 65 at R2 + 1.5x volume + MACD down

### Filters
- ✅ No-Trade Zone (Yellow) - Skip choppy ranges
- ✅ Volume Filter (Green/Red) - Require conviction
- ✅ MACD Direction - Must align with bias
- ✅ RSI Extreme - For momentum confirmation
- ✅ VIX Level - Adjust stops if > 25

### Exit Rules
- Time-based: Close after 2 hours no movement
- RSI-based: Exit when opposite extreme hit
- Level-based: Exit at next major level
- Signal-based: Opposite signal = immediate close
- Volume-based: Exit if volume collapses

## HOW TO USE (3-STEP QUICK START)

### STEP 1: Deploy to TradingView
```
1. Open TradingView → Create new indicator
2. Copy code from: NIFTY_Pro_Trading_System_v2.pine
3. Paste into Pine Script editor
4. Save & Add to Chart
```

### STEP 2: Configure Settings
```
Pivot Type: Camarilla (for NIFTY)
RSI Length: 14
MACD: 12, 26, 9 (defaults)
Volume MA: 20
VIX Filter: ON (threshold 25)
Display: All options ON
```

### STEP 3: Start Trading
```
Watch for:
1. Horizontal lines (R1, R2, S1, S2)
2. Green arrow = BUY signal
3. Red arrow = SELL signal
4. Yellow zone = NO TRADE zone
5. Set SL immediately, hold for target
```
