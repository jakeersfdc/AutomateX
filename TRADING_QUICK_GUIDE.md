## 🟢 When to BUY (BUY Signal)

### Check Dashboard
```
✓ Signal = "BUY" (Green box)
✓ Confidence > 80%
✓ Entry Level shown
✓ Stop Loss shown
✓ Risk:Reward > 1:2
```

### Execute Trade
```
1. Check chart - make sure Entry Level is current resistance
2. Place BUY order
   - Entry: At resistance (or slightly above)
   - Quantity: Based on your stop loss risk
   - Stop Loss: Exact level shown on dashboard
   - Target: Use Target 1, 2, 3 levels

3. Position Management
   - 50% profit booking at Target 1
   - 30% hold till Target 2
   - 20% aggressive at Target 3 (or full exit at TP)
```

### Example Trade
```
Symbol: NIFTY
Signal: BUY at 24,025
Stop Loss: 23,970
Targets:
  - T1: 24,075 (50 points) - Exit 50%
  - T2: 24,110 (85 points) - Exit 30%
  - T3: 24,200 (175 points) - Exit 20%

Risk: 55 points (Entry - SL)
Reward: 175 points (Entry - Target)
RR: 1:3.2 (Excellent!)

💰 If lot size = 50 shares
   Risk per trade = 55 × 50 = ₹2,750
   Max profit = 175 × 50 = ₹8,750
```

### What It Means
```
Signal = "WAIT"
❌ Don't trade yet!
⏳ Wait for better setup
🔄 Market consolidating
📊 Indicators misaligned
```

### What to Do
```
1. Monitor the chart
2. Wait for signal to change to BUY or SELL
3. Don't force entry when signal is WAIT
4. Use this time to prepare (risk calculator, position size)
```

## 💡 Smart Trading Rules (MUST FOLLOW)

### Rule 1: Signal + Confirmation
```
Never trade on signal alone!
Wait for:
  ✓ Price above/below entry level
  ✓ Volume confirmation
  ✓ Second alert or indicator
```

### Rule 2: Risk Management (Most Important!)
```
Never risk more than 2% per trade!

Example:
  Your Account: ₹1,00,000
  Max Risk per Trade: 2% = ₹2,000
  
  Stop Loss Distance: 50 points
  Lot Size = ₹2,000 / 50 = 40 shares
```

### Rule 3: Always Use Stop Loss
```
NO EXCEPTIONS!
ALWAYS place SL at exact level shown on dashboard.
Never move stop loss against you.
```

### Rule 4: Book Profits
```
Don't be greedy!
Follow exit plan:
  T1 (50%): Book when target reached
  T2 (30%): Hold for more
  T3 (20%): Let it run
```

### Rule 5: Trade What You See
```
Signal = BUY  → Trade BUY
Signal = SELL → Trade SELL
Signal = WAIT → Don't trade
```

### Rule 6: Follow the Trend
```
BUY only when:
  ✓ EMA Alignment is Bullish (20 > 50 > 200)
  ✓ Price above all EMAs
  ✓ Volume increasing

SELL only when:
  ✓ EMA Alignment is Bearish (20 < 50 < 200)
  ✓ Price below all EMAs
  ✓ Volume increasing
```

## 🎲 Position Size Calculator

```
Formula:
Position Size = Risk Amount / Stop Loss Distance

Example 1: BUY Signal
  Account: ₹5,00,000
  Risk per trade: 1% = ₹5,000
  Entry: 24,025
  SL: 23,970
  Distance: 55 points
  
  Position Size = 5,000 / 55 = 90 shares
  
  Maximum loss: 90 × 55 = ₹4,950 (acceptable)

Example 2: SELL Signal
  Account: ₹5,00,000
  Risk per trade: 2% = ₹10,000
  Entry: 23,950
  SL: 24,020
  Distance: 70 points
  
  Position Size = 10,000 / 70 = 142 shares
  
  Maximum loss: 142 × 70 = ₹9,940 (acceptable)
```

```
Result: Pyramid losses
Solution: Cut losses, don't add
```

## 📞 Troubleshooting

### Dashboard shows "WAIT" but I want to trade
```
❌ Don't trade! Wait for BUY or SELL
Reason: Not enough confluence
Solution: Set alert and check back later
```

### Got BUY signal but price is falling
```
❌ This is normal!
Reason: Confirmation candle not yet formed
Solution: Wait for entry level + volume confirmation
```

### Entry level keeps changing
```
✅ This is fine!
Reason: Dashboard updates in real-time
Solution: Use latest level from dashboard
```

### Got 5 BUY signals today
```
⚠️  Take only best signals
Quality > Quantity
Skip signals with <85% confidence
Maximum 3 trades per day
```
