## ⚠️ Troubleshooting

### Issue: "Can't push to GitHub"
```
Solution: 
1. Delete c:\Users\Jakeer Hussain\Profitforce\.git folder
2. Run git commands again
3. If still stuck: reinstall Git from git-scm.com
```

### Issue: "Deployment failed"
```
Solution:
1. Check Procfile exists and is correct
2. Check requirements.txt
3. View Railway logs for errors
4. Redeploy from Railway dashboard
```

### Issue: "Dashboard showing blank"
```
Solution:
1. Hard refresh browser (Ctrl+Shift+Del)
2. Wait 30 seconds
3. Try again
4. Check browser console (F12) for errors
```

### Issue: "Can't see signals"
```
Solution:
1. Wait for first data to load (30 seconds)
2. Check internet connection
3. Verify API endpoint is responding
4. Check market trading hours (9:15-15:30 IST)
```
