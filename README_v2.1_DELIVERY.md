### **3 New Professional Indicators**

#### 1. ☁️ **Ichimoku Cloud** (STRONGEST)
- **Score Impact**: +3 (Maximum strength)
- **Signals**: Above bull cloud (+3), Below bear cloud (-3), TK cross (±2)
- **Advantage**: All-in-one visual system, works all timeframes
- **Best For**: Determining market direction at a glance

#### 2. ⚡ **Stochastic RSI** (FASTEST)
- **Score Impact**: +2 for oversold/overbought
- **Signals**: Catches oversold 2-3 bars BEFORE regular RSI
- **Advantage**: Faster entries on reversals
- **Best For**: Intraday scalping and day trading (5-15 min)

#### 3. 📊 **Rate of Change - ROC(9)** (MOMENTUM FILTER)
- **Score Impact**: +2 for strong moves, kills false signals when near 0%
- **Signals**: ROC > 1.5% = strong bull, ROC < -1.5% = strong bear
- **Advantage**: Prevents whipsaw entries when momentum is dead
- **Best For**: Filtering weak signals in range-bound markets

## 🎛️ **SIGNAL SCORING SYSTEM** (NEW)

### How It Works
```
Maximum Score: ±11 points (all indicators aligned)
Minimum to Trade: ±3 points

Score Breakdown:
├─ Ichimoku Cloud: ±3 (strongest)
├─ Stochastic RSI: ±2 (fast entries)
├─ ROC: ±2 (momentum confirmation)
├─ RSI: ±1 (basic momentum)
├─ MACD: ±1 (trend direction)
├─ Volume: ±1 (conviction)
└─ VIX: ±1 (volatility adjustment)
```

### Example Scores

**Strong Buy Signal (Score: 7+)**
```
✅ Ichimoku:      +3 (above bull cloud)
✅ StochRSI:      +2 (oversold K<20, D<20)
✅ ROC:           +2 (ROC = 2.1%)
✅ RSI:           +1 (RSI = 28)
✅ MACD:          +1 (histogram green)
Result: 🟢 BUY | Confidence: 95%
```

**Weak Signal - Skip (Score: 2)**
```
⚠️ Ichimoku:      +1 (TK cross only)
⚠️ StochRSI:      +1 (K>D mid-zone)
❌ ROC:           0 (ROC = 0.1% DEAD)
⚠️ MACD:          +1 (barely green)
Result: 🟡 WAIT | Confidence: 20%
```
