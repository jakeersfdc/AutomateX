## 📊 NEW INDICATORS AT A GLANCE

### ☁️ ICHIMOKU CLOUD (+3 Points)
```
🟢 ABOVE BULL CLOUD + Tenkan > Kijun = STRONGEST BUY
🔴 BELOW BEAR CLOUD + Tenkan < Kijun = STRONGEST SELL
🟡 IN CLOUD = NO TRADE (consolidation)

TK CROSS (Tenkan crosses Kijun) = +2 entry signal
Chikou > Price (26 bars back) = +1 confirmation
```

### ⚡ STOCHASTIC RSI (+2 Points)
```
K < 20 & D < 20 = OVERSOLD = 📉 BUY setup
K > 80 & D > 80 = OVERBOUGHT = 📈 SELL setup
K > D = Bull cross = +1 momentum up
K < D = Bear cross = +1 momentum down
```

### 📊 RATE OF CHANGE (+2 Points)
```
ROC > +1.5% = STRONG momentum up = BUY ✅
ROC < -1.5% = STRONG momentum down = SELL ✅
ROC near 0% = MOMENTUM DEAD = 🚫 AVOID ENTRY!
ROC > 0.5% = Weak bull
ROC < -0.5% = Weak bear
```

## 💰 POSITION SIZING FORMULA

```
Account Risk = 1% of account
Account Size: ₹500,000
Account Risk: ₹5,000

Trade:
Entry Price: ₹22,000
SL Price: ₹21,900 (100 pts)
Risk per trade: ₹5,000

Quantity = Risk ÷ (SL - Entry)
         = 5,000 ÷ 100
         = 50 lots (5 contracts × 10 lots each)

TP at +200 pts = ₹22,200 = +₹10,000 profit (2:1 reward)
```
